# For developers

### \<Field />

Internally, all form elements rely on the component \<Field />. The html element
to render can be defined with the \`type\` props and must be one of \`text\`,
\`checkbox\`, \`radio\`, \`input\`, \`select\`.

```jsx
import { Field } from "../components/field";

<Field
  type="radio"
  chartId="AHhgGxoZRC"
  path={"height"}
  label="variable"
  value="http://..."
/>;
```

### useField()

This hook handles form events and dispatches an action to update the application
state stored in local storage.
